Programming
在 Linux 终端当中，请输入查询 `python` 命令对应的是哪个文件夹里的 `python` 的命令.
Back: 
```shell
which python
```
Addition: 
这个命令在 Python 开发很常用，在虚拟环境较多的情况下，可以判断命令程序的位置.
