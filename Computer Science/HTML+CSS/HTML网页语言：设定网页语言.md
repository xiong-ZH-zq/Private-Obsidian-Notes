Programming
请将下列的 HTML 网页代码补全使得网页语言为简体中文：
```html
<!DOCTYPE html>
<html _____>
....
</html>
```
Back: 
```
lang="zh-CN"
```
Addition:
`zh` 表示中文，`zh-CN` 表示大陆地区中文. 这个代码叫作 ISO 代码，在必要的时候可以查阅手册.
