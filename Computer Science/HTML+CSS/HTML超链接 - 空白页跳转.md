请写 HTML 超链接标签，链接文字为搜索，跳转链接为 Google ，且为新开一个空白页签跳转.
Back: 
```html
<a href="https://www.google.com" target="_blank">搜索</a>
```
Addition: 
与一般的不同的地方就在于 `target` 属性的设置. 如果是当前页面，可以直接设为 `_self` ，但是默认就是 `_self` 所以没有必要写出来.
