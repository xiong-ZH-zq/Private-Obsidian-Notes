Programming
请写出 shell 命令取出 `a.txt` 中的命令**追加到** `b.txt` 内容的后部.
Back: 
```shell
cat < a.txt >> b.txt
```
Addition: 
`>>` 表示追加而非覆盖. 在这种输入输出流里面，如果没有使用这些运算符重新接线，就会直接输出到终端.
