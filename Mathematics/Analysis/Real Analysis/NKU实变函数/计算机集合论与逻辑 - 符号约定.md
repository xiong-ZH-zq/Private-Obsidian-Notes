| 符号                     | 含义                                      |
| ---------------------- | --------------------------------------- |
| $A\sim B$              | $A$ 和 $B$ 的差集                           |
| $X^k$                  | $k$ 个集合 $X$ 的直积                         |
| $\omega$               | 自然数集                                    |
| $X^{<\omega}$          | $\displaystyle\bigcup_{k\in \omega}X^k$ |
| $\mathrm{Dom}(F)$      | $F$ 的定义域                                |
| $\mathrm{Im}(F)$       | 函数 $F$ 的值域                              |
| $\mathrm{At}(\varphi)$ | 语句 $\varphi$ 中的 $L_0$ 语句.               |
| $\mathrm{TA}$          | 真值指派函数全体组成的集合                           |
